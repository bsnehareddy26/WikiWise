From Wikipedia, the free encyclopedia
This page is referenced in the Wikipedia Glossary.

On Wikipedia, renaming might refer to:

  • Moving a page to a different name; most users can do this, via the Move tab (you must be autoconfirmed to have a Move tab)
    • If you can't do this on a page you wish to move (note that some pages are move-protected), you can make a request at Wikipedia:Requested moves.
  • Renaming a category; this is generally done by experienced users, and normally requires prior discussion at Categories for discussion.
    • To make a request, go to Categories for discussion.
  • Moving a file to a different name; only a few users ("administrators" and "file movers") can do this
    • To make a request, use the template {{Rename media}} (see template for instructions).
  • Changing a username to a different name; only a handful of users ("Stewards and Global renamers") can do this
    • To make a request, go to Wikipedia:Changing username.
  • Renaming a Wikipedia page section
    • If you can't think of a better name yourself you may use the {{Rename section}} template

If you need any help, see Help:Contents and, if necessary, ask for help at the Teahouse.
